Expand description

Describes an implementation of a connected datagram socket.

Rather than using Socket for datagram-oriented sockets, the DatagramSocket trait purposely does not implement AsyncRead/AsyncWrite, which are traits with stream semantics. For example, the AsyncReadExt::read_exact method which issues as many reads as possible to fill the buffer provided.

For a similar reason, std::net::UdpSocket does not implement io::Read nor does [tokio::net::UdpSocket] implement [tokio::io::AsyncRead].
